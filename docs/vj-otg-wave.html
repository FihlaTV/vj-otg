vj-otg-wave

Custom element for distorting the coordinate using a wave pattern.

<vj-otg-wave t="[number]" speed="[number]" frequency="[number]" amplitude="[number]">

Properties:
Name Type Description
t number

time offset increase this to move the wave.
speed number

how fast the waves travel affects t.
frequency number

the amount of waves
amplitude number

the size of the waves.
Source:
  • visuals/distorts/wave.js, line 20

Example

Create a continuous moving wave by using the time uniform in the t property.

<vj-otg-time-uniform></vj-otg-time-uniform>
<vj-otg-wave t="[time]" frequency="3.0" amplitude="0.05" speed="0.5"></vj-otg-wave>
