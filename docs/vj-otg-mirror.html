vj-otg-mirror

Custom element for mirroring the current corrdinate.

<vj-otg-mirror mode="[number]">

Properties:
Name Type Description
mode number

value from 0-3 describing how the mirror is done

  • 0, No mirror
  • 1, Left to Right
  • 2, Top to Bottom
  • 3, Left to Right and Top to Bottom
Source:
  • visuals/distorts/mirror.js, line 14

Example

<vj-otg-mirror mode="1"></vj-otg-mirror>
