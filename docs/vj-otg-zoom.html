vj-otg-zoom

Used to zoom and pan the current coordinate.

<vj-otg-zoom factor="[number]" position="[vector2]">

Properties:
Name Type Description
factor number

how far to zoom in, 1.0 is normal, 0.5 is zoomed out 2.0 is zoomed in.
position vector2

The place to zoom on, 0.5 0.5 is the center.
Source:
  • visuals/distorts/zoom.js, line 17

Example

Zoom in 2x

<vj-otg-zoom factor="2" position="0.5, 0.5"></vj-otg-zoom>
