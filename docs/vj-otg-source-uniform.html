vj-otg-source-uniform

Used for defining a special type of uniforms. Textures. These can be accessed using the texture sampler in the glsl code.

<vj-otg-source-uniform name="[number]" src="[querySelector]" index="[number]">

Properties:
Name Type Description
name number

name of the variable in glsl
src querySelector

element to use as texture, can be <img> or <video>
index number

index for retrieving texture using getSource(index, coordinate); in the glsl code
Source:
  • visuals/uniforms/source.js, line 18

Examples

Import an image, give it the name minnie.

<vj-otg-source-uniform src="#minnie" name="minnie" index="1"></vj-otg-source-uniform>

Use it in GLSL for a static texture.

gl_FragColor = texture2D(minnie, newUV);

Use it in GLSL for a dynamic texture.

gl_FragColor = getSource(1, newUV);
