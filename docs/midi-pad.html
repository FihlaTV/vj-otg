midi-pad

Used to read or emulate a midi pad.

<midi-pad channel-press="[number]" channel-release="[number]" note="[number]" value="[number]">

Properties:
Name Type Description
channel-press number

Channel press events are emitted on
channel-release number

Channel release events are emitted on, usually 16 lower
note number

The note to listen to.
value number

The value emited if the emulated pad is clicked
Source:
  • midi-pad.js, line 68

Example

Access a Midi Pad

<midi-controller>
	<midi-pad channel-press="144" channel-release="128" note="36" value="127" id="pad1">PAD1</midi-pad>
</midi-controller>
